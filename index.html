Data Correlator

It’s time to explore the world’s data! Data Correlator allows comparison and interaction with data order to gain a deeper understanding of its impact. The data sets used compare area (in km²), GDP (in the international dollar), and population for countries around the world, along with the cities in the word that have populations exceeding 1 million people.

Data Correlator is a data visualization tool to be used for understanding and comparing data sets. Data Correlator is built on Javascript, HMTL, SCSS, and uses the D3.js library.
